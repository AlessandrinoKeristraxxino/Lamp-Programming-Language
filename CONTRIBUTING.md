# Contributing to LAMP

Thank you for your interest in contributing to LAMP! Here's how to get started.

---

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/lamp.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Build and test: `make`
6. Commit: `git commit -m "feat: description of your change"`
7. Push: `git push origin feature/your-feature-name`
8. Open a Pull Request

---

## Commit Style

Please follow this format:

| Prefix | Use for |
|--------|---------|
| `feat:` | New feature |
| `fix:` | Bug fix |
| `docs:` | Documentation changes |
| `refactor:` | Code refactoring |
| `test:` | Adding tests |

---

## Code Style

- Use C++17 or later
- Keep code readable and well-commented
- Follow the existing file structure

---

## Reporting Bugs

Open an issue on GitHub with:
- A clear description of the bug
- Steps to reproduce it
- Expected vs actual behavior
- Your OS and compiler version

---

## Questions?

Open a GitHub Discussion or an issue labeled `question`.
